# Risen Six

> Six strings. Endless rise.

An online guitar curriculum — beginner through advanced — with theory baked into every lesson and a separate path for those who want to study CAGED, the circle of fifths, and the rest of it on its own. A venture by **CW Custom Guitars**.

## Stack (planned)

- **Frontend:** Vite + React + TypeScript + Tailwind CSS
- **Hosting:** Cloudflare Pages (free, custom-domain ready)
- **Auth + DB + Storage:** Supabase (free tier — Postgres, auth, file storage)
- **Source control:** GitHub
